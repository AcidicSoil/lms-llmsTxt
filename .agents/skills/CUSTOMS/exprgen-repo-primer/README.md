# exprgen-repo-primer

A combined Agent Skill bundle for scanning a repository and drafting `.exprgen.yaml` recipes before development begins.

This combines three candidate skill ideas:

- Exprgen Config Scout
- Repo Expression Primer
- Exprgen Recipe Generator

## Contents

- `SKILL.md` - Main skill instructions.
- `references/config-generation.md` - Recipe rules and generation heuristics.
- `templates/exprgen-config.yaml` - Starter output template.
- `scripts/scan-repo.sh` - Optional local repository summary helper.

## Install

Unzip this folder into your Agent Skills directory as `exprgen-repo-primer/`.
