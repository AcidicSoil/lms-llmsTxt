# production-workflow-runbook

Agent Skill for generating production-ready E2E delivery runbooks across conventional software targets.

## Contents

- `SKILL.md` — activation instructions and routing
- `references/core-lifecycle.md` — canonical lifecycle
- `references/project-overlays.md` — brownfield and greenfield overlays
- `references/production-readiness.md` — release gates and done criteria
- `references/output-templates.md` — reusable runbook templates
- `references/targets/` — target-specific runbooks

## Install

Copy the `production-workflow-runbook/` directory into the agent skills directory used by your runtime.
