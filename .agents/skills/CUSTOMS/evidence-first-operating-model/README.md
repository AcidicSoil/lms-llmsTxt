# Evidence-First Operating Model Skill

A bundled Agent Skill combining six workflow patterns into one reusable assistant operating model:

1. Repository/state consolidation
2. Skill-guided planning
3. Documentation/memory audit
4. Evidence-first execution
5. Correction responsiveness
6. Safe artifact consolidation

Use it for bounded, autonomous cleanup or planning passes where the assistant must inspect evidence, apply local instructions, act safely, validate, report uncertainty, and leave a clean handoff.
