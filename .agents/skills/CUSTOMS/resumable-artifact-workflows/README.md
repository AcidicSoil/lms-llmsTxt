# Resumable Artifact Workflows

A language- and project-agnostic Agent Skill for hardening batch automation that reads manifests, processes many items, writes artifacts, and needs reliable resume/delta/backfill behavior.

Start with `SKILL.md`. Detailed patterns live in `references/`, reusable JSON/TSV/layout starters live in `templates/`, and `scripts/delta_index.py` is an optional generic helper.
