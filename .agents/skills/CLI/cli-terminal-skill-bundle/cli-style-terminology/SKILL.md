# CLI Style Terminology

## Workflow

1. Identify the visible CLI traits: color, spacing, tables, panels, aligned columns, borders, and help formatting.\n2. Name the style using terms such as ANSI-styled terminal output, structured CLI output, modern terminal-native CLI UX, or enhanced CLI help formatting.\n3. Clarify whether the user means a standard CLI, shell output, or a full-screen TUI.\n4. Provide short terminology options plus one recommended phrase.

## Output Requirements

- Use language-agnostic terminology unless the user specifies a stack.
- Distinguish standard CLI output from full-screen TUIs.
- Prefer concise, implementation-ready language.
- Include the phrase "modern terminal-native CLI with structured, ANSI-styled output" when it fits.
