# Gemini Cli

> Gemini CLI aims to bring powerful AI capabilities — including multimodal understanding, large context windows, and tool integration — directly into the terminal for developers. It empowers users to analyze codebases, automate workflows, generate content, and collaborate with AI in a lightweight, extensible, and secure environment. The project supports both personal and enterprise use cases, with options for free tier access, API key management, or Vertex AI integration, while maintaining an open-source Apache 2.0 license.

**Remember:**
- Gemini CLI: Terminal-based AI agent for Google's Gemini models
- MCP (Model Context Protocol): Framework for extending Gemini with custom tools and integrations
- Built-in Tools: File operations, shell commands, web search, and more
- Checkpointing: Save and resume long-running AI conversations
- Custom Commands & Extensions: User-defined commands via TOML files
- Trusted Folders: Secure execution policies by folder
