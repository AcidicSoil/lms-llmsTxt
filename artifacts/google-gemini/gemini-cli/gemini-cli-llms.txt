# Gemini Cli

> Gemini CLI aims to empower developers by bringing advanced AI capabilities — including multimodal reasoning, code generation, and contextual grounding — directly into their terminal environment. It enables automation of complex tasks (like code reviews or file operations), supports custom integrations via MCP servers, and provides enterprise-grade features like sandboxing, trusted folders, and telemetry. The project is designed for both individual developers and teams, offering flexible authentication options and a rich ecosystem of tools and extensions.

**Remember:**
- Gemini CLI: Terminal-based AI agent powered by Gemini models
- MCP (Model Context Protocol): Framework for extending Gemini CLI with custom tools
- Checkpointing: Save and resume long-running conversations
- Shell Commands & Built-in Tools: Execute system commands, file operations, and web searches
- Custom Skills & Extensions: User-defined commands and behaviors
- Trusted Folders: Control execution policies by folder
