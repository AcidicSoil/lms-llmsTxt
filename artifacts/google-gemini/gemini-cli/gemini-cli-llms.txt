# Gemini Cli

> The purpose of the Gemini CLI is to provide developers with a lightweight, terminal-based interface to interact with Googleâ€™s Gemini AI models. It enables users to query, generate, debug, and automate tasks using natural language, supported by built-in tools (file operations, shell commands, web search) and extensible MCP server integrations. The project aims to empower developers to work more efficiently in their command line environments, with features like checkpointing, token caching, and GitHub integration for automated workflows.

**Remember:**
- Terminal-first AI agent
- Gemini 3 models with 1M token context
- MCP (Model Context Protocol) for custom tool integrations
- Built-in tools: file operations, shell commands, web search
- Checkpointing and resumable sessions
- Context files (GEMINI.md) for persistent project settings
