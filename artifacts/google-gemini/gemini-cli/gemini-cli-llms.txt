# Gemini Cli

> Gemini CLI is an open-source AI agent that brings the power of Google’s Gemini model directly into the terminal for developers. It enables lightweight, interactive or scripted access to Gemini’s capabilities—including code generation, debugging, automation, and real-time information retrieval—while supporting extensibility via MCP servers and custom tools.

**Remember:**
- Gemini Model Access
- Terminal-First Interface
- MCP Server Integration
- Built-in Tools (File System, Shell, Web Fetch)
- Context Files (GEMINI.md)
- Checkpointing & Session Persistence
