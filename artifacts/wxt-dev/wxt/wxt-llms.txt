# Wxt

> WXT aims to simplify the development of web extensions by providing a next-generation, framework-agnostic toolkit that abstracts away boilerplate and browser-specific complexities. It enables developers to build, test, and publish extensions efficiently using modern tools like TypeScript, Vite, and automated workflows. The framework supports both MV2 and MV3 standards, ensuring compatibility across browsers while offering rapid development cycles through hot-reload and modular architecture.

**Remember:**
- Web Extension Framework
- MV2 & MV3 Compatibility
- TypeScript Support
- Auto-Imports
- Frontend Framework Agnostic
- Vite-based Build System
