# Cliproxyapi

> CLIProxyAPI serves as a universal proxy for CLI-based AI models, allowing users to access OpenAI, Gemini, Claude, Codex, Qwen, and iFlow through OAuth without exposing API keys. It supports multi-account load balancing, streaming responses, multimodal input, and integrates with popular tools like Amp CLI, VSCode extensions, and desktop apps. The project empowers developers to build custom AI tools and dashboards that abstract away the complexity of multiple LLM providers.

**Remember:**
- CLI Proxy
- OAuth Authentication
- Multi-Account Load Balancing
- Model Mapping
- Translator Layer
- Executor Layer
