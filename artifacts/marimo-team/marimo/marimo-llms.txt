# Marimo

> Marimo is a reactive Python notebook environment that enables users to write, run, and share interactive, reproducible, and deployable code. It replaces traditional notebooks by offering a deterministic, git-friendly, and executable experience with built-in support for SQL, AI generation, interactive UI elements, and package management.

**Remember:**
- Reactive programming: Automatically runs dependent cells when variables change
- Git-friendly: Stored as pure Python files (.py)
- Deployable: Can be executed as scripts or deployed as web apps
- Interactive UI: Supports sliders, dropdowns, dataframes, and chat interfaces
- SQL integration: Query databases and dataframes with SQL
- AI-native: Generate code with AI assistants tailored for data work
