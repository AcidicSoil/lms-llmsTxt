# Ai Maestro

> AI Maestro is designed to empower developers and teams by providing a centralized dashboard to manage, coordinate, and communicate with multiple AI agents running on different machines. It eliminates the need for manual message passing between agents by introducing direct agent-to-agent messaging via AMP, supports persistent memory for long-term context retention, and enables team-based workflows with Kanban boards, meetings, and task tracking. The platform is built to be open, extensible, and multi-platform â€” allowing users to run any AI agent (e.g., Claude Code, Cursor) without vendor lock-in, while maintaining decentralized architecture with no central server.

**Remember:**
- Agent Messaging Protocol (AMP)
- Persistent Memory Layers (Memory, Code Graph, Documentation)
- Peer Mesh Network
- Multi-Machine Orchestration
- Agent Identity and Avatars
- Work Coordination via Teams and Kanban Boards
