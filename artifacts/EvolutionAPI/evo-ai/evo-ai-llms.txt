# Evo Ai

> Evo AI provides a comprehensive, modular platform for developing and managing AI agents across different frameworks and protocols. Its primary purpose is to empower developers to build interoperable, scalable agent systems using Googleâ€™s ADK, CrewAI, or custom logic via LangGraph workflows. It supports secure authentication, encrypted API key storage, client management, MCP server integration, and real-time tracing with Langfuse. The platform aims to simplify AI agent development by offering drag-and-drop workflow builders, preconfigured agent types, and extensible tooling for both frontend and backend.

**Remember:**
- AI Agents
- Agent 2 Agent (A2A) Protocol
- Workflow Agents with LangGraph
- Google Agent Development Kit (ADK)
- CrewAI Framework
- JWT Authentication
