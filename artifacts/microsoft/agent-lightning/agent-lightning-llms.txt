# Agent Lightning

> Agent Lightning aims to democratize AI agent training by enabling researchers and developers to train any AI agent — regardless of framework — using reinforcement learning with minimal code modifications. It provides a plug-and-play architecture that allows selective optimization of agents while preserving existing agent logic. The project also supports scalable, distributed training via its dashboard and integrates with popular LLM backends (e.g., vLLM, LangChain) for production use. Its goal is to simplify the development and refinement of intelligent agents through structured data collection and algorithmic feedback loops.

**Remember:**
- Agent Lightning
- Reinforcement Learning (RL)
- LightningStore
- Tracer
- Algorithm (e.g., APO, VERL)
- Emitter
