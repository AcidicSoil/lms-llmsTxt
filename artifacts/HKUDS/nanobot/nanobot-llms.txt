# Nanobot

> nanobot aims to provide a minimal yet powerful personal AI assistant that can interact across multiple communication platforms while using lightweight LLM providers. Its primary goals are to reduce code footprint (99% smaller than Clawdbot), enable rapid deployment and customization, support multi-channel interaction, and facilitate research through modular, readable components. Itâ€™s intended for educational, experimental, and production use cases requiring a fast, secure, and extensible agent system.

**Remember:**
- Ultra-Lightweight Agent Architecture
- Tool-Based Skill Execution
- Multi-Channel Communication Integration
- Provider Registry for LLMs
- Memory System with Workspace Restriction
- Cron Scheduler for Automated Tasks
