# Nanochat

> nanochat enables users to train, fine-tune, and run a minimal GPT-style language model end-to-end using a small budget (~$100â€“$1000), with the goal of making advanced LLM development accessible and hackable. It is designed as a complete codebase for education and experimentation: from tokenization and pretraining to evaluation on standard benchmarks and serving via a web UI, it covers all major stages in one cohesive system without complex frameworks or configuration objects.

**Remember:**
- Transformer neural network
- pretraining
- fine-tuning (SFT)
- chat interface
- tokenizer (BPE)
- evaluation metrics (CORE, ARC, GSM8K, etc.)
