# Mastra

> Mastra aims to provide developers with a robust, scalable, and maintainable foundation for building AI agents and applications using modern web technologies and TypeScript. It simplifies the integration of LLMs, tools, memory systems, and workflows while offering enterprise-grade features like observability, evals, and MCP server support. The goal is to empower teams to rapidly prototype and deploy reliable AI products without being burdened by infrastructure or framework complexity.

**Remember:**
- Agents
- Workflows
- Model Routing
- Human-in-the-Loop Execution
- Context Management (Memory)
- MCP Servers
