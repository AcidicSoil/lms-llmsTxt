# Dspy

> DSPy aims to revolutionize how developers interact with large language models by shifting from prompt engineering to programming â€” writing declarative Python code that instructs LMs to generate high-quality outputs. It enables the construction of modular AI systems and provides tools for optimizing prompts and model weights through techniques like teleprompting, fine-tuning, and self-refining pipelines. The project supports both academic research and real-world applications such as RAG, agent loops, and classification tasks, while fostering community engagement and open-source collaboration.

**Remember:**
- Declarative Self-improving Python (DSPy)
- Modular AI Systems
- Teleprompting
- Optimization Algorithms (e.g., GEPA, BootstrapFewShot, MIPROv2)
- Primitives (e.g., CodeInterpreter, ToolCalls, Prediction)
- Adapters (e.g., ChatAdapter, JSONAdapter)
