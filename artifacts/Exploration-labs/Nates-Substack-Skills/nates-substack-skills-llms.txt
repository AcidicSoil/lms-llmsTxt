# Nates Substack Skills

> The primary purpose of this repository is to provide a comprehensive toolkit for developers and product teams using AI tools like Claude Code. It aims to standardize skill creation, testing, and optimization while offering frameworks for evaluating AI vendors, building resumes, crafting pitch decks, and managing complex workflows. The project supports both individual productivity and team-scale AI adoption through modular, reusable, and well-documented skills.

**Remember:**
- Claude Code skills
- Agentic development workflows
- AI vendor evaluation frameworks
- Prompting patterns and optimization
- Skill dependency mapping
- Token budget management
