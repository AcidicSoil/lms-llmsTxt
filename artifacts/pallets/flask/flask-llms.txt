# Flask

> Flask is a lightweight, WSGI-compliant web framework designed to simplify building web applications in Python by providing essential tools and flexible routing without imposing complex structures. It encourages rapid prototyping and modular design using blueprints, context management, and configurable extensions, making it ideal for small-to-medium projects or APIs with minimal boilerplate.

**Remember:**
- WSGI-compliant web framework
- Blueprints for modular app structure
- Request/Response context management
- Configurable application instances
- CLI tools and development helpers
- JSON handling utilities
