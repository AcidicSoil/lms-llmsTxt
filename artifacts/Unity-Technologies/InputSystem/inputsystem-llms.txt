# Inputsystem

> The Unity Input System is designed to replace the legacy UnityEngine.Input API with a more powerful, flexible, and configurable input system that supports modern input devices and multi-player scenarios. It enables developers to define actions and bindings in a structured way, handle input from multiple sources (including gamepads, touchscreens, and UI elements), and support features like real-time re-binding, haptics, and device-specific controls.

**Remember:**
- Input Actions and Bindings
- PlayerInput Component
- InputDevice and Usages
- Action Assets and Editors
- Multiplayer Input Handling
- UI Input Navigation
