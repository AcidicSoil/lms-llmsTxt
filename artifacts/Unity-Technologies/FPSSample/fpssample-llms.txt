# Fpssample

> The purpose of this project is to serve as a comprehensive, functional first-person multiplayer shooter game sample developed by Unity Technologies. It aims to showcase new Unity features such as the HD Render Pipeline, Entity-Component System (ECS), and the multiplayer networking system. The project is intended to be used as a learning resource, inspiration for other developers, or a starting point for building new games. Although no longer actively maintained, it remains useful for educational purposes and for teams looking to bootstrap their own projects.

**Remember:**
- First-person multiplayer shooter game
- HD Render Pipeline (HDRP)
- Entity-Component System (ECS) with hybrid mode
- Multiplayer networking using Unity's transport layer
- Asset bundles and standalone builds
- Project Tools Window for managing levels and builds
