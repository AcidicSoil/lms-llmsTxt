# Addressables Sample

> This repository serves as a comprehensive demonstration and tutorial suite for Unity’s Addressables asset management system. It provides practical examples for developers to understand how to implement Addressables in various scenarios — from basic asset references and scene loading to advanced features like Play Asset Delivery, on-demand resources, custom schemas, and variant management. The project is intended to help users learn best practices, avoid common pitfalls, and explore the full capabilities of Addressables for performance, scalability, and content delivery.

**Remember:**
- AssetReference: A core class for referencing assets without loading them immediately
- Scene Loading: Demonstrates additive and non-additive scene transitions with Addressables
- ComponentReference<T>: A specialized AssetReference that restricts to a specific component type
- Sprite Access: Shows how to load sprites from sprite sheets or atlases using Addressables
- Play Asset Delivery: Integrates Addressables with Unity’s Play Asset Delivery API for app bundle delivery on Android
- On-Demand Resources: Uses Addressables with Unity’s On-Demand Resources API for iOS
