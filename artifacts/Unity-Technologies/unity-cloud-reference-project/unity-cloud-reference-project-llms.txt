# Unity Cloud Reference Project

> The Unity Cloud Reference Project was intended to serve as a customizable, extensible template for developers building collaborative 3D design review tools within Unity. It provided pre-built components for VR, annotation, presence, asset streaming, and multi-user collaboration, enabling industry-specific applications to be developed faster by leveraging Unityâ€™s cloud services.

**Remember:**
- Collaborative 3D design review tool
- Unity Cloud integration (presence, annotation, asset streaming)
- VR and AR support
- Customizable navigation modes (walk, fly, teleport, VR)
- Tool extensibility (annotation, asset list, presence, profile tools)
- Multi-language localization
