# Com.Unity.Multiplayer.Samples.Coop

> The Boss Room sample is designed to educate developers on multiplayer networking patterns using Unity Netcode for GameObjects. It demonstrates how to build a co-op RPG with features like character abilities, animations, object replication, session management, authentication, and scene transitions. The sample aims to serve as a reference for building multiplayer games by showcasing best practices and common patterns used in the industry.

**Remember:**
- Server-authoritative gameplay with latency-masking animations
- NetworkTransform for synchronized position and rotation
- RPCs for quick actions (e.g., mage bolts)
- NetworkAnimator for animation control
- Client-driven input anticipation for action timing
- NetworkObjectSpawner for dynamic object placement
