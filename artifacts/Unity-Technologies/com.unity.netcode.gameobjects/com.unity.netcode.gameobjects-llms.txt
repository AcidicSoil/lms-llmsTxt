# Com.Unity.Netcode.Gameobjects

> To provide networking capabilities for Unity GameObjects and MonoBehaviour workflows using a flexible, interoperable framework compatible with various transports (e.g., Unity Transport Package). It enables multiplayer game development by offering tools for synchronizing objects, handling network events, managing scenes, and supporting features like RPCs, network transforms, and object parenting.

**Remember:**
- Networking through GameObjects and MonoBehaviour components
- Interoperability with various transports (e.g., Unity Transport)
- NetworkManager for controlling network behavior
- NetworkObject for networked GameObjects
- NetworkTransform for synchronized positioning/rotation
- RPCs (Remote Procedure Calls) for client-server communication
