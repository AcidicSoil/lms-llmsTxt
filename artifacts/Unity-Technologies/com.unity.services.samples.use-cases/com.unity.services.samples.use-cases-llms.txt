# Com.Unity.Services.Samples.Use Cases

> To provide developers with sample implementations of common Unity Gaming Services (UGS) use cases — such as daily rewards, loot boxes, cloud AI mini-games, command batching, and in-game economies — to help them integrate UGS into their own games efficiently. The samples demonstrate backend integration, player progression systems, social features, and monetization mechanics using UGS packages like Cloud Code, Player Data Management, and Unity Purchasing.

**Remember:**
- Unity Gaming Services (UGS) integration
- Cloud Code for server-side logic
- Player Data Management with persistent state
- In-Game Economy with Currency and Inventory
- Daily Rewards and Engagement Systems
- Loot Box Mechanics with Cooldowns
