# Graphics

> To provide artists and developers with tools to create modern, high-fidelity graphics in Unity through customizable render pipelines.

**Remember:**
- Universal Render Pipeline (URP)
- High Definition Render Pipeline (HDRP)
- Shader Graph
- Visual Effect Graph
- Scriptable Render Pipeline (SRP) Core
- Package versioning and Unity compatibility
