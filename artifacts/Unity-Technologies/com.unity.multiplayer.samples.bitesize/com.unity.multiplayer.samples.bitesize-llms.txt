# Com.Unity.Multiplayer.Samples.Bitesize

> The purpose of this repository is to provide educational and practical samples for developers learning to implement multiplayer functionality in Unity using Netcode for GameObjects (NGO). It serves as a starting point for understanding core networking concepts such as distributed authority, client-driven movement, dynamic prefab systems, and common multiplayer patterns.

**Remember:**
- Netcode for GameObjects (NGO)
- Multiplayer Topologies (e.g., Distributed Authority)
- Network Variables
- Object Pooling
- Client-Driven Movement
- Dynamic Addressables Network Prefabs
