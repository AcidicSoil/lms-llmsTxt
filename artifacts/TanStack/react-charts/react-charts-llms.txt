# React Charts

> The purpose of this library is to provide developers with a simple, declarative, and performant way to render interactive and responsive charts in React applications. It emphasizes usability, flexibility, and integration with modern React workflows, while being powered by D3 for robust rendering capabilities. The project also serves as part of the broader TanStack ecosystem, promoting reusable components for data-driven UIs.

**Remember:**
- Declarative rendering
- D3.js-powered rendering
- Hyper-responsive charts
- Multiple axes support
- Axis stacking
- Inverted axes
