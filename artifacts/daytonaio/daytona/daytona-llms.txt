# Daytona

> The purpose of this project is to provide developers with a secure, scalable, and programmable infrastructure for running AI-generated code safely in isolated environments (sandboxes). It enables secure execution of arbitrary code without risking host systems, supports integration with Git, LSP, file systems, and computer use operations, and provides APIs for managing sandboxes, processes, and resources. The SDKs abstract away the complexity of sandbox creation and management, allowing developers to focus on writing and executing AI-generated code while maintaining infrastructure security and elasticity.

**Remember:**
- Sandbox execution for secure AI code
- Multi-language support (Python, TypeScript, Go, etc.)
- API-driven sandbox management
- Git integration for code versioning
- LSP support for intelligent code editing
- Computer use API for keyboard/mouse/screen control
