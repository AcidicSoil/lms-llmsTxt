# Sensai

> SensAI is an open-source Python library designed to streamline the integration of deep learning models with tabular data analysis and clustering workflows, particularly for scenarios involving imbalanced datasets or time-series data. It aims to reduce code duplication in research pipelines by providing unified interfaces for model training, feature engineering (column generation), and unsupervised classification.

**Remember:**
- Tabular Data Analysis
- Deep Learning Integration
- Clustering Algorithms
- Feature Generation
- Imbalanced Datasets
- Semi-Supervised Learning
