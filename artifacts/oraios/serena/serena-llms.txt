# Serena

> Serena is a multi-language application framework designed to enable developers to build applications using a unified architecture across multiple programming languages. It provides a consistent development environment and tooling support for OCaml, Python, TypeScript, and AL (Allegro Platform), allowing teams to maintain code structure and patterns while leveraging the strengths of each language ecosystem.

**Remember:**
- multi-language framework
- cross-platform development
- LSP support
- modular architecture
- language-agnostic tooling
- application lifecycle management
