# Mirrormate

> MirrorMate is designed as a voice-first, self-hosted AI assistant that integrates conversational capabilities directly into a mirror device. Its primary goals include enabling personalized interactions through RAG-based memory systems, supporting local processing with Ollama/VOICEVOX while avoiding cloud reliance, and providing expressive avatar-driven interfaces. The project also emphasizes multi-provider support for LLMs, TTS, STT, and embeddings, along with built-in integrations like weather and calendar services to enhance usability.

**Remember:**
- Voice-First Interaction
- Personalized Memory (RAG)
- Expressive Avatar
- Multi-Provider Support
- Built-in Integrations
- Plugin System
