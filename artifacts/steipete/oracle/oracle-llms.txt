# Oracle

> Oracle enables developers to bundle prompts and files for processing by multiple AI models, facilitating context-aware responses across different platforms. Its primary goals are to simplify prompt engineering, support multi-model experimentation, and provide fallbacks (like browser automation or manual copy) when API access is unavailable. It also offers session persistence, remote execution, and integration with developer tools like Cursor and Codex.

**Remember:**
- Prompt bundling
- Multi-model API runs
- Browser automation
- Session management
- Remote browser service (oracle serve)
- Model selection strategy
