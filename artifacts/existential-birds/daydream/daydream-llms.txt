# Daydream

> Daydream is a Python-based framework designed to evaluate and generate responses from the "Daydream" AI model. It facilitates various testing scenarios—such as structured output, streaming, and tool use—while providing a unified CLI entry point and fixture generation capabilities for data augmentation.

**Remember:**
- Daydream AI Model
- Structured JSON Output
- Tool Use Testing
- Streamed Responses
- CLI Interface
