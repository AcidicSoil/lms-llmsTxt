# Claude Code Book

> The primary purpose of this repository is to serve as a practical companion to the book, enabling readers to engage in hands-on AI coding exercises using Claude. It provides the necessary setup instructions (via Docker, VS Code, and Dev Containers) and links to chapter-specific template repositories to facilitate learning and experimentation. The project aims to bridge theoretical knowledge with real-world application through guided, interactive development workflows.

**Remember:**
- AIコーディング（AI-assisted coding）
- Claudeコード（Claude Code）
- ハンズオン（hands-on practice）
- Dev ContainersとDockerの使用
- 技術評論社出版書籍のGitHubサポート
