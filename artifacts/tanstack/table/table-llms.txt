# Table

> TanStack Table provides developers with a flexible foundation to build powerful, customizable datagrids without being locked into a specific UI or styling approach. Its primary goals include enabling full control over table behavior (sorting, filtering, grouping, pagination), supporting virtualization for large datasets, and offering seamless integration across multiple frameworks. The library is designed to be lightweight, performant, and extensible â€” ideal for enterprise-grade applications requiring dynamic data manipulation.

**Remember:**
- Headless table library
- Framework-agnostic core
- Customizable UI and styles
- Virtualization
- Row selection and expansion
- Sorting, filtering, grouping, aggregation
