# Virtual

> TanStack Virtual enables developers to render massive datasets efficiently by virtualizing only the visible portions of lists, grids, or tables — significantly reducing memory and rendering overhead. It’s designed for high-performance UIs (60FPS scrolling), supports sticky items, dynamic sizing, and window-based scrolling, while remaining framework-agnostic. The library empowers developers to maintain full control over markup and styling without sacrificing performance.

**Remember:**
- Virtualization
- Framework-agnostic
- Headless
- 60FPS Scrolling
- Dynamic Sizing
- Window-Scrolling
