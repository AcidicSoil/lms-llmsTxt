# Adk Python

> The Agent Development Kit (ADK) empowers developers to construct, evaluate, and deploy AI agents with flexibility and control. It simplifies the process of building agent workflows — from basic tasks to complex systems — using Python, while remaining model-agnostic and deployment-agnostic. The framework supports tool integration, multi-agent orchestration, code execution in secure environments, and agent configuration without coding, making it ideal for both prototyping and production use. Its modular design encourages extensibility and community contributions, with tools for evaluation, debugging, and deployment via cloud services like Vertex AI.

**Remember:**
- Agent Development Kit (ADK)
- Code-First Development
- Tool Confirmation (HITL)
- Multi-Agent Systems
- A2A Protocol Integration
- Agent Config (No-code Agent Definition)
