# Adk Js

> The ADK aims to empower developers to create flexible, testable, and deployable AI agents by providing a TypeScript-based framework that allows direct code definition of agent behavior, tool usage, and orchestration. It supports integration with Google Cloud services and offers a development UI for debugging and evaluation. The project also enables multi-agent systems and interoperability via the A2A protocol, targeting both local and cloud environments.

**Remember:**
- Agent
- Tool
- LLM Processor
- Code Executor
- Memory Service
- Plugin System
