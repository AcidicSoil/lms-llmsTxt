# Adk Go

> The ADK for Go enables developers to construct sophisticated AI agents using Goâ€™s concurrency and performance advantages, with support for tool integration, session persistence, and deployment across cloud environments. It simplifies agent development by providing reusable components like workflow agents, tool managers, and LLM processors while maintaining flexibility for custom logic and model agnosticism. The framework is particularly suited for building scalable, deployable agent systems in cloud-native architectures.

**Remember:**
- Agent Workflow Orchestration
- Tool Integration (Function Tools, Artifact Loading)
- Session Management with Persistence
- LLM Processing and Streaming Support
- Modular Architecture with Separation of Concerns
- Code-First Development Paradigm
