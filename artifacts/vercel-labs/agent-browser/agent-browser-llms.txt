# Agent Browser

> agent-browser empowers AI agents to automate web tasks with precision, using a fast Rust CLI that interfaces with Chromium via Playwright. It enables AI-driven workflows such as navigating sites, interacting with elements using semantic or ref-based selectors, and capturing snapshots for deterministic LLM interactions. The tool supports isolated sessions, network interception, authentication handling, and real-time streaming for human-AI collaboration â€” all while being cross-platform and performant.

**Remember:**
- Headless browser automation
- Semantic locators (ARIA roles, text, labels)
- Snapshot-based element refs (@e1, @e2)
- Agent mode with JSON output for AI integration
- Daemon-client architecture (Rust CLI + Node.js daemon)
- CDP mode for remote browser control
