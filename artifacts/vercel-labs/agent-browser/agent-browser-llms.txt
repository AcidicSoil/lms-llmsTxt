# Agent Browser

> The project aims to empower AI agents with reliable, fast, and scalable browser automation capabilities. By providing a native Rust CLI with rich command support (including semantic selectors, session management, and cloud providers), it enables agents to navigate, interact, and extract data from websites programmatically â€” especially suited for tasks like form automation, UI testing, or agent-driven web browsing. It also supports persistent state across sessions and integrates with cloud browser infrastructure for serverless or CI environments.

**Remember:**
- Agent Mode
- Semantic Locators (refs)
- Snapshot Tree
- Session Isolation
- Persistent Profiles
- CDP Connection
