# Agent Browser

> The purpose of agent-browser is to empower AI agents with a fast, reliable, and extensible browser automation tool that operates in headless mode while supporting persistent sessions, authentication, and cloud-based execution. It enables agents to navigate websites, interact with elements using semantic locators or deterministic refs, manage cookies and storage, and even stream browser views for human-AI collaboration. The tool is optimized for AI workflows — such as parsing snapshots, identifying interactive elements, and executing actions — while offering fallbacks for environments without native binaries.

**Remember:**
- Ref-based element selection
- Semantic locators (role, text, label, etc.)
- Session persistence with profiles and encryption
- Snapshot accessibility tree for AI interpretation
- CDP mode for remote browser control
- Streaming for live preview or pair browsing
