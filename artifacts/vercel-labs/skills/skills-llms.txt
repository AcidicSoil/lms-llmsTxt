# Skills

> The purpose of this project is to provide a unified CLI tool that simplifies the management of reusable Agent Skills across multiple AI coding agents. It enables developers to discover, install, update, and remove skills tailored to specific agent capabilities — such as generating PRs, integrating with external tools, or automating workflows — while maintaining compatibility and modularity. The tool supports both project-local and global installations, integrates with various agent ecosystems, and provides robust discovery and management features for developers and teams.

**Remember:**
- Agent Skills
- SKILL.md files
- YAML frontmatter (name, description, metadata)
- Installation scope (project vs. global)
- Symlink vs. copy installation methods
- Supported agents (e.g., Claude Code, Cursor, OpenCode)
