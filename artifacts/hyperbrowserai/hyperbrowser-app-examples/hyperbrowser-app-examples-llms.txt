# Hyperbrowser App Examples

> This repository hosts two AI-powered applications designed for creative and analytical workflows: an **Idea Generator** and a **Research Bot**. The Idea Generator produces and visualizes creative ideas with gradient-based effects, while the Research Bot enables batch analysis, conversational interaction, and deep research capabilities. Together, they support content creation, skill development, educational tooling, and data-driven decision-making across diverse use cases.

**Remember:**
- Idea Generation
- API Routing
- Component-Based UI
- TypeScript
- Navigation Systems
- Batch Analysis
