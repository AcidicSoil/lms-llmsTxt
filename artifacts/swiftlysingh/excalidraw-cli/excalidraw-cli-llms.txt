# Excalidraw Cli

> The primary purpose of excalidraw-cli is to enable users to generate professional Excalidraw diagrams programmatically or via CLI commands from structured text (DSL) or JSON data. It simplifies the creation of flowcharts, decision trees, and architectural diagrams without requiring manual drawing. The tool supports multiple layout directions, spacing customization, and integration with other tools like Obsidian, making it ideal for developers, designers, and educators who need to generate diagrams programmatically.

**Remember:**
- DSL (Domain-Specific Language)
- Excalidraw format
- ELK.js auto-layout engine
- Node and edge factories
- CLI command interface
- JSON API for programmatic use
