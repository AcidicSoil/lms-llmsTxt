# A2A

> The A2A Protocol aims to break down silos between AI agents by establishing a standardized, open communication framework that allows agents to discover each otherâ€™s capabilities, negotiate interaction modes, collaborate on complex tasks, and operate without exposing internal logic or memory. It promotes an interoperable, secure, and enterprise-ready ecosystem for agentic applications, encouraging innovation through community-driven standards.

**Remember:**
- Agent2Agent (A2A) Protocol
- Agent Cards
- JSON-RPC 2.0 over HTTP(S)
- Agent Discovery
- Streaming and Async Communication
- Opaque Agent Collaboration
