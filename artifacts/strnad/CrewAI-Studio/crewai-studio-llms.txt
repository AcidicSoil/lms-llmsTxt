# Crewai Studio

> CrewAI Studio aims to democratize access to AI agent orchestration by providing an intuitive, no-code interface for users unfamiliar with programming. It enables users to define and execute multi-agent workflows (crews) using pre-built or custom tools, with support for diverse LLM backends and real-world interactions. The project also encourages community contribution through open-source development, sponsorship, and documentation.

**Remember:**
- CrewAI Studio
- Streamlit GUI
- Multi-agent orchestration (Crews)
- LLM Providers (OpenAI, Groq, Anthropic, etc.)
- Custom Tools (API calls, file writing, code interpreter, web scraping)
- Knowledge Sources
