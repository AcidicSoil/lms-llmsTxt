# Lobster

> Lobster provides a structured, typed workflow shell for AI agents to execute complex, composable tasks — such as monitoring GitHub PRs, triaging emails, or executing system commands — without generating verbose or token-heavy queries. It enables resumable and deterministic execution by leveraging JSON pipelines and state management. The project aims to reduce cognitive load on agents by abstracting repetitive operations into reusable macros, while maintaining local-first execution and avoiding ownership of OAuth tokens.

**Remember:**
- Typed JSON-first pipelines
- Local-first execution
- Composable macros for AI agents
- Approval gates (e.g., `approve`)
- Workflow files (YAML/JSON with steps, env, condition)
- State management and resumability
