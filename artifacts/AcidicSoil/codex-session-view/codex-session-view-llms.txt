# Codex Session View

> Codex Session Viewer is designed to empower developers and AI engineers to deeply inspect, replay, and debug interactive user sessions with precision. By offering a rich timeline view, filtering capabilities, and AI-powered analysis (via the “Session Coach” module), it helps identify misalignments, track conversational context, and generate actionable remediation prompts. The tool supports session persistence, repo metadata grouping, and integrates with modern tooling like TanStack, shadcn/ui, and Browser Echo for robust client-side debugging.

**Remember:**
- Session Replay
- Timeline View
- Event Filtering (ACTION, MUTATION, LOG)
- Repo-aware Session Explorer
- Chat Dock & Discovery Panel
- Misalignment Detection & Remediation
