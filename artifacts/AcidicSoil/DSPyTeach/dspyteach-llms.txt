# Dspyteach

> To create a CLI tool that analyzes source files using DSPy, generates structured teaching materials for developers, and supports customizable prompts for both teaching briefs and refactor templates.

**Remember:**
- DSPy integration for multi-step reasoning
- File analysis with teaching brief generation
- Refactor template creation mode
- CLI interface for code analysis
- Support for multiple language models (Ollama, LM Studio)
- Structured markdown output with dspy.Refine
