# Codex Skills

> The primary purpose of this repository is to showcase how to build and deploy skills for `codex-cli`, enabling users to teach the system specialized tasks such as document creation, animated GIF generation, or enterprise communication. It provides a practical, open-source reference for developers to understand skill architecture, including metadata, scripting, and file format handling. The repository also includes production-grade document skills (like docx, pdf, pptx) that power Claude’s document features, offering insight into handling complex binary formats. It aims to inspire and guide users in creating their own skills, whether for personal automation or enterprise-scale AI workflows.

**Remember:**
- Skills
- modular
- self-contained folders that extend codex-cli’s capabilities
- SKILL.md
- the core metadata and instruction file for each skill
- containing YAML frontmatter and markdown instructions
