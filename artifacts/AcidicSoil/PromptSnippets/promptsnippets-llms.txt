# Promptsnippets

> PromptSnippets aims to reduce repetitive typing by enabling users to define customizable snippets that can be triggered with a single keystroke (e.g., typing `/` followed by `Tab` to navigate variables). It emphasizes ease of use, privacy (no login required), and flexibility through variable snippet support. The project also includes documentation and a playground for demonstrating usage, targeting developers and content creators who value efficiency in their workflow.

**Remember:**
- Variable Snippets
- Chrome Extension
- Content Script
- Zustand State Management
- UI Components (e.g., Popup, Settings Panel)
- Snippet Editor & Import/Export
