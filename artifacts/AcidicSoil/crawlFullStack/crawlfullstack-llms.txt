# Crawlfullstack

> The primary purpose of this repository is to provide a flexible, configurable, and safe web crawler capable of generating offline Markdown mirrors of websites. It enables developers to crawl any documentation-style site (via seed URLs) or legacy platforms (via predefined profiles), while preserving deterministic link structures, supporting image downloads, and validating generated links. The tool emphasizes safety through rate limits and robots.txt compliance, and extensibility via profile-based configurations. Additionally, it supports advanced features like TypeScript integration, CI/CD workflows, and end-to-end testing for robust deployment.

**Remember:**
- Config-first crawling engine
- Generic seed mode vs legacy site profiles
- Deterministic link rewriting and reindexing
- Playwright-based browser automation
- Robots.txt compliance and rate limiting
- Markdown conversion pipeline with frontmatter
