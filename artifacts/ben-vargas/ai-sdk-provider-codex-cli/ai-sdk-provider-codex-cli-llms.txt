# Ai Sdk Provider Codex Cli

> This project provides a community-driven bridge between the Vercel AI SDK and OpenAI’s Codex CLI, allowing developers to integrate high-end GPT-5 models into their applications via Node.js. It enables features like streaming text, object generation with Zod schemas, tool execution monitoring, and multimodal image input—all while supporting flexible logging, configuration, and fallback mechanisms for local or CI environments. The provider ensures compatibility with legacy models and future Codex CLI features while offering granular control over model behavior, sandboxing, and tool invocation.

**Remember:**
- Codex CLI
- AI SDK v6 LanguageModelV3
- GPT-5.1 / GPT-5.2 models
- Streaming text and object generation
- JSON Schema validation via `--output-schema`
- Tool streaming with real-time tool-call and result events
