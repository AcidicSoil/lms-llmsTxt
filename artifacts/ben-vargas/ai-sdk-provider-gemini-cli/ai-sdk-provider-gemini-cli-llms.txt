# Ai Sdk Provider Gemini Cli

> This project provides a seamless, type-safe interface to Google’s Gemini models through the Vercel AI SDK, allowing developers to leverage Gemini’s capabilities — including text generation, tool calling, structured output via Zod, and multimodal inputs — in their applications. It supports multiple authentication methods (OAuth and API key), customizable configurations, and advanced features like streaming responses and logging control. The project is intended for community use and is not affiliated with Google or Vercel.

**Remember:**
- AI SDK Provider
- Gemini Models (e.g., gemini-3-pro-preview)
- OAuth Authentication via Gemini CLI
- Zod Schema Mapping for Structured Output
- Tool Calling and Function Invocation
- Streaming Responses
