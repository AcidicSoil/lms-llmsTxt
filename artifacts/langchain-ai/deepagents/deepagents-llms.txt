# Deepagents

> Deep Agents aims to simplify the creation and deployment of autonomous AI agents by providing a ready-to-run, configurable harness with built-in planning, context management, file system access, sub-agents, and sandboxed executionâ€”all based on LangGraph. It enables developers to quickly prototype or deploy agents with minimal setup, while supporting customization for specific use cases, model providers, and integration with sandbox platforms (Modal, Runloop, Daytona).

**Remember:**
- Agent harness
- Prompt engineering
- Tool integration
- Sub-agents
- Context management
- LangGraph
