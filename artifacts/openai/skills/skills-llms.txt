# Skills

> The primary purpose of this repository is to provide a standardized, modular system for AI agents to discover and utilize reusable skills in Codex. It enables developers to extend Codex’s functionality by installing pre-packaged skills that handle specific tasks — such as meeting intelligence, research documentation, or spec-to-implementation — through a consistent interface. The project also supports skill creation and distribution via a skill creator and installer, promoting community-driven innovation and repeatable workflows.

**Remember:**
- Agent Skills
- Codex
- Curated Skills
- Experimental Skills
- Skill Installer
- Skill Creator
