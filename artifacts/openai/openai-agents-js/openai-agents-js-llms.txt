# Openai Agents Js

> The OpenAI Agents SDK aims to simplify the development of complex multi-agent systems by providing a lightweight, provider-agnostic framework that integrates LLMs with tools, guardrails, handoffs, and tracing. It enables developers to build workflows across various environments (Node.js, Deno, Bun) and supports advanced features like voice agents, parallel execution, and human-in-the-loop interactions.

**Remember:**
- Agents
- Handoffs
- Guardrails
- Tracing
- Tools/Function Integration
- Multi-Agent Workflows
