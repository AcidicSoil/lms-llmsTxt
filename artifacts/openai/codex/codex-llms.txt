# Codex

> The purpose of Codex CLI is to provide a locally installed, command-line interface for OpenAI’s Codex agent — enabling developers to generate and execute code directly on their machines. It supports integration with ChatGPT plans (Plus, Pro, Team, Edu, Enterprise) and offers a terminal-based experience with features like live execution, file search, markdown rendering, and sandboxing for secure local runs. The CLI also allows users to interact with models via structured prompts, support for slash commands, and advanced UI components such as transcript copy, status indicators, and feedback selection.

**Remember:**
- Local code generation and execution
- Terminal-based user interface (TUI)
- Integration with ChatGPT plans and API keys
- Sandboxing for secure local runs
- Live execution with streaming responses
- Markdown rendering and diff visualization
