# Codex

> The purpose of this repository is to provide a locally installed, command-line interface (CLI) agent called Codex that allows users to interact with AI models for code generation, debugging, and editing. It enables developers to run Codex locally on their machines, offering enhanced control, privacy, and performance compared to cloud-based alternatives. The project supports:  - Interactive terminal sessions via TUI. - Integration into IDEs (like VS Code or Cursor). - Desktop app experience through `codex app`. - Collaboration modes for multi-agent workflows. - Authentication via ChatGPT accounts or API keys. - Secure sandboxing environments (including Windows support).  The goal is to empower developers with a locally running, customizable, and highly capable AI assistant for code-related tasks.

**Remember:**
- Codex CLI: The main command-line interface for interacting with the local AI agent
- TUI (Terminal User Interface): Interactive UI for terminal-based Codex usage
- Sandboxing: Secure execution environments for models, including Windows and Linux support
- Multi-Agent Collaboration: Support for multiple agents working together in a session
- Markdown and Code Rendering: Live rendering of markdown and code blocks with syntax highlighting
- ChatGPT Integration: Authentication via ChatGPT accounts or API keys
