# Crewai

> CrewAI is designed to empower developers and enterprises to build intelligent, autonomous AI automation systems with precision and flexibility. Its purpose is to provide a lean, high-performance framework for orchestrating multi-agent systems — whether through collaborative Crews or precise Flows — enabling complex enterprise workflows while maintaining simplicity and control. It targets both simple tasks and large-scale, production-grade automation, with features like human-in-the-loop support, telemetry, and scalable architecture.

**Remember:**
- Crews: Autonomous teams of AI agents that collaborate and delegate tasks dynamically
- Flows: Event-driven workflows with granular control over execution paths and state management
- Multi-Agent Orchestration: Combining autonomous agent behavior with precise workflow control
- LLM Integration: Flexible support for various language models, including local models via Ollama or LM Studio
- Telemetry & Observability: Anonymous data collection to improve the framework’s features
- Memory Management: Short-term and long-term memory support for context retention across tasks
