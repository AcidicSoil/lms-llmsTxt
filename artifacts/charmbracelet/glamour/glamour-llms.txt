# Glamour

> Glamourâ€™s primary purpose is to enable developers to render Markdown content in the terminal with rich styling, such as colors, fonts, and layouts, using ANSI escape codes. It empowers CLI applications to display beautifully formatted documentation, output, or user interfaces. The library supports both predefined styles (e.g., dark, dracula, light) and custom stylesheet creation, making it flexible for diverse terminal environments.

**Remember:**
- ANSI escape codes
- Markdown rendering
- Styling via JSON/Go style files
- Auto-style detection (dark/light)
- TermRenderer with configurable options
- Custom renderer and environment-based styling
