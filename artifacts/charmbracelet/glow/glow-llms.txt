# Glow

> Glow aims to bring visual appeal and usability to markdown rendering in the terminal by offering a high-performance TUI and CLI interface. It enables users to explore local or remote markdown content interactively, with customizable styles, word wrapping, and pager support. The project seeks to improve developer productivity and documentation consumption within the CLI environment while maintaining compatibility across multiple platforms and terminals.

**Remember:**
- Terminal-based markdown reader
- Text User Interface (TUI)
- Command-line interface (CLI)
- Styling via JSON or YAML
- Word wrapping and paging
- Git repository integration
