# Freeze

> Freeze enables developers to visually capture code snippets or terminal outputs as static images, ideal for documentation, presentations, or sharing code states across platforms. It supports both static code rendering and dynamic terminal output via the `--execute` flag, with rich customization options including themes, fonts, borders, shadows, and window controls. The tool also integrates with configuration files and interactive modes to streamline user experience.

**Remember:**
- Code screenshot generation
- Terminal output capture
- Customization via flags and config files
- Interactive TUI mode
- Syntax highlighting with themes
- Font embedding and styling
