# Ollama Studio

> Ollama Studio aims to empower users to easily build and deploy custom AI models using Ollama’s local inference engine through an intuitive web interface. It simplifies the process of uploading system prompts, selecting base models, adjusting temperature settings, and launching new models or engaging in model-based debates—all while abstracting away complex dependency management via Pinokio. The tool is designed for developers and enthusiasts who want to personalize AI behavior without deep technical setup.

**Remember:**
- Gradio UI
- Ollama Models
- System Prompts
- Temperature Control
- Pinokio Installer
- GGUF Models
