# Assistant Ui

> assistant-ui aims to empower developers to build scalable, production-ready AI chat experiences in React apps with minimal boilerplate. It provides reusable, composable primitives for messages, threads, attachments, tool calls, and more â€” all with built-in streaming, auto-scrolling, keyboard shortcuts, and accessibility. The library supports multiple AI backends and optional cloud features like persistence and analytics, making it flexible for both small projects and enterprise deployments.

**Remember:**
- Composable primitives
- Streaming chat UI
- AI SDK integration
- LangGraph support
- Assistant Cloud (managed backend)
- Tool calls and JSON rendering
