# Suggested Commands
- Create venv: `python3 -m venv .venv && source .venv/bin/activate`
- Or with uv: `uv venv && source .venv/bin/activate`
- Install editable package with dev deps: `uv pip install -e '.[dev]'`
- Run tests: `pytest -q`
- Run a focused test file: `pytest -q tests/test_analyzer.py`
- Run smoke test similar to CI: `uv run --isolated --no-project --with dist/*.whl --with pytest pytest -q tests/smoke_test.py`
- Build package: `uv build`
- Run CLI locally: `lmstxt <github_repo_url>`
- Run MCP server: `lmstxt-mcp`
- Run HyperGraph UI dev server: `npm --prefix hypergraph run dev`
- Root helper alias for UI dev: `npm run ui:dev`
- Useful Linux shell commands in this environment: `git status`, `git diff`, `ls`, `find`, `grep`, `cd`, `python -m pytest`.
- Verification note: no dedicated repo eslint or TypeScript typecheck config was found at the root; root `package.json` is script-oriented, so Python verification is the primary completion check unless a task explicitly touches HyperGraph.
